## =============== Local Installation ===============
### Git Clone
* git clone https://github.com/nikdval/absenteeApp

### Install react and libraries packages
*  Navigate to folder */absenteeApp*
*  npm install

### Run the App
#### ======== Developer Mode =========
* mpm run dev
* Open the browser to localhost:8080
